# CISSP 复习助手

一个专为 CISSP (Certified Information Systems Security Professional) 考试复习设计的助手应用。

## 项目简介

CISSP 复习助手旨在帮助考生系统化地准备 CISSP 认证考试，提供以下功能：

- 📚 知识域管理
- 📝 练习题和模拟考试
- 📊 学习进度跟踪
- 🎯 重点知识点复习
- 📈 学习数据分析

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI**: React 18

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。

### 3. 构建生产版本

```bash
npm run build
npm start
```

## 项目结构

```
cissp-study-assistant/
├── app/              # Next.js App Router 页面
├── components/       # React 组件
├── lib/             # 工具函数和配置
├── public/          # 静态资源
└── types/           # TypeScript 类型定义
```
